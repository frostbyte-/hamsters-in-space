Game Setup

Solo / Pass-and-Play

Online Multiplayer

Each player joins on their own device. Names are set in the Playroom lobby.

Each turn: Production → Action → Resupply

Actions: Deploy (play cards to tableau for resources), or use suit abilities:
⚡ Engine=Steal
💥 Weapon=Attack
🧭 Navigation=Sneak
🛡️ Shield=Block

Buy cards using resources (⚙️ Scrap, 🧵 Tech) acquired

Build 3 same-suit tableau cards into a ship part (🚀)

Work with Pirates (👾) to get access to parts that are cheap (but will they work?)

First player to build 3 ship parts wins!

wins!
